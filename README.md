# Kanbas Learning Management System

A full-stack LMS built with Node.js, React, and MongoDB supporting 200+ users.

## Architecture
- **Frontend:** React.js, NextJS
- **Backend:** Node.js, Express, MongoDB

## Features
- User authentication and authorization
- Course management and analytics
- Real-time updates
